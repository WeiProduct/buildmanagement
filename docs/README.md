# AI建管系统 支持网站

这是AI建管系统的官方支持网站，使用GitHub Pages托管。

## 网站内容

- **首页** - 产品介绍和支持选项
- **常见问题** - 用户常见问题解答
- **隐私政策** - 详细的隐私保护说明
- **服务条款** - 使用条款和协议

## 访问地址

网站将托管在: https://weiproduct.github.io/buildmanagement/

## 部署说明

1. 将docs文件夹推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择从`docs`文件夹部署
4. 等待几分钟后即可访问

## 更新内容

要更新网站内容，直接编辑相应的HTML文件并推送到仓库即可。
